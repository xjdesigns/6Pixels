Selectlist w/ Search(NG2)

Angular 2 Working

Selectlists are custom components that allow for use of a custom select with search capabilites.

There are specific requirements for its use so refer to the element table for details.

ARIA-LABEL Aria is set using the name property for accessibility.

When using the custom component if you flag multiple as true you MUST pass an array of objects with matching properties you specify to currentSelection. If you set multiple to false your currentSelection is a string.
