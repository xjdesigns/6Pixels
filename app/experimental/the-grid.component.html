The Grid Layout

Need help understanding the new display grid rule. Just edit the grid in the sidebar and apply the changes to it so you can see how it will work.

Lets learn the grid. This is a basic tool for learning the grid. NOTE: This does not use template-areas and some rules can be shorthand.

This is the grid you have defined::

    display: grid;
    grid-template-columns: {{tempColumns}};
    grid-template-rows: {{tempRows}};
    grid-column-gap: {{gap.column}};
    grid-row-gap: {{gap.row}};
    justify-items: {{justifyItem}};
    align-items: {{alignItem}};
    justify-content: {{justifyContent}};
    align-content: {{alignContent}};
