Guidelines

6PIXELS is meant to be a next gen CSS UI FRAMEWORK / PATTERN LAB, and can present progressive enhancement. Please use the @supports. Progressive enhancement code should live at the top of your SCSS file.

Everything should be created with MOBILE FIRST in mind.

Everything should be in increments of 6 PIXELS. This creates specific patterns and expectations.

Wih any new elements/components please supply a new route, along with at least one live example and code example.
